# 수익화 블로그 완전 가이드 (알파남/애드센스팜 자료 종합정리)

## PART 1. 수익 구조 이해

### 1-1. 구글 애드센스 수익 원리
- 구글 애드센스는 **CPC(클릭당 비용)** 및 **CPM(1000회 노출당 비용)** 기반 수익 모델
- 2024년부터 **CPC → CPM 전환** 추세 (CPM은 1000회 노출당 수익)
- 수익 배분: 광고 수익의 **68%**가 퍼블리셔(블로거)에게 지급, 32%는 구글 몫
  - 예: 클릭 1만회 발생 시 6,800원 수익, 3,200원은 구글
- CPC는 키워드마다 다름: **$0.01 ~ $30** 범위
- 핵심 지표:
  - **RPM (Revenue Per Mille)**: 1,000 페이지뷰당 예상 수익
    - 공식: `RPM = (예상 수익 / 페이지뷰) * 1000`
    - 예: 25페이지뷰, 수익 $0.15 → RPM = $6
  - **CPC (Cost Per Click)**: 광고 1회 클릭당 단가
  - **CTR (Click Through Rate)**: 클릭률 (노출 대비 클릭 비율)
    - 예: 1,000회 노출에 2회 클릭 → CTR 0.2%

### 1-2. 수익 실적 사례 (알파남)
- 2020년 시작 → 2개월차 300만원, 2개월차 700만원, 3개월차 900만원, 3개월차 2,000만원
- 2022년 기준: 783개 글, 월 조회수 23,479 → 월 수익 31,716,254원
- 티스토리 53개 블로그 운영, 1450개 글 → 월 수익 1000만원 이상
- 핵심: **50개 키워드만 잘 잡으면 500~1000만원 가능**

### 1-3. 네이티브 광고 (애드센스 외 수익원)
- **Taboola**: RPM 800~1,200원 달성 사례
- **Aceplanet**: RPM 500~4,000원
- 애드센스 + 네이티브 광고 병행 시 **월 6,000원 이상 RPM** 가능
- 2023년 기준 월 1,500~2,000만원 수익 (네이티브 광고 포함)
- 2023년 1~8월: 네이티브 광고 수익 6,095만원

## PART 2. 플랫폼 선택

### 2-1. 네이버 vs 티스토리 vs 워드프레스 비교

| 항목 | 네이버 | 티스토리 | 워드프레스 |
|------|--------|----------|-----------|
| 진입장벽 | 낮음 | 중간 | 높음 |
| SEO | 네이버 내부만 | 구글 SEO 유리 | SEO 최강 |
| 수익화 | 애드포스트(낮음) | 애드센스 가능 | 애드센스+다양한 광고 |
| 커스터마이징 | 제한적 | 중간 | 무제한 |
| 비용 | 무료 | 무료 | 호스팅비(월 5~30달러) |
| 소유권 | 카카오 소유 | 카카오 소유 | 본인 소유 |

### 2-2. 추천 성장 경로
```
초보 (0~300글) → 티스토리로 시작, 100개 글 3개월 목표
중급 (300~1000글) → 티스토리 + 워드프레스 병행
고급 (1000글 이상) → 워드프레스 중심, 다중 사이트, 외주, 유료광고
```

### 2-3. 전문 블로그 vs 잡블로그
- **전문 블로그**: 하나의 주제에 집중
  - 장점: 전문성 인정, 구글 E-E-A-T 신뢰도 UP, 충성 독자 확보, 수익 안정적, 체류시간 높음
- **잡블로그**: 여러 주제 혼합
  - 장점: 트렌드 대응 빠름, 초보자 진입 쉬움
  - 단점: 전문성 낮음, 구글 평가 불리
- **결론**: 전문 블로그 5개(3개 전문, 2개 잡블로그) 운영이 이상적
  - 전문 블로그: 1~1년반 동안 꾸준히 키워야 효과
  - 잡블로그: 시즌 키워드 대응용

## PART 3. 티스토리 블로그 세팅

### 3-1. 기본 개설
1. tistory.com 가입
2. 블로그 이름: **키워드 포함** (예: "건강정보닷컴")
3. 스킨 선택: **poster** 또는 **bookclub** 추천
4. 주소 설정: 키워드 관련 영문명 (변경 불가하므로 신중하게)

### 3-2. 필수 세팅 7단계
1. **블로그 이름**: 키워드 포함 + 짧게
2. **설명(Description)**: SEO 키워드 포함, 80~150자
3. **카테고리**: 6개 내외로 체계적 구성
4. **대표 이미지(파비콘)**: 브랜딩용
5. **개인 도메인**: 2차 도메인 연결 (가비아에서 구매)
6. **RSS/사이트맵**: 검색엔진 등록용
7. **프로필**: 전문성 표현

### 3-3. 광고 배치 (HTML 코드)
- HTML 편집 → `s_article_rep` 태그 찾기
- `<section id="container">` 바로 아래에 광고 코드 삽입

**1안 - 단일 광고:**
- 본문 시작 전 위치에 배치

**2안 - 더블 광고 (PC 2개, 모바일 1개):**
```html
<!-- PC: 2개 나란히, 모바일: 1개만 표시 -->
@media (min-width: 680px) → 300x250 2개
@media (min-width: 768px) → 336x280 2개
```

**본문 중간 광고:**
- `_article_rep_desc_` 태그 활용
- 본문 내용과 자연스럽게 어울리는 위치에 삽입

### 3-4. GA4 (구글 애널리틱스 4) 연동
1. Google Analytics 계정 생성 → 측정 ID 발급
2. 티스토리: 관리 → 꾸미기 → 스킨 편집 → HTML → `<head>` 태그 안에 GA4 코드 삽입
3. 데이터 활용: 페이지뷰, 체류시간, 이탈률, 유입경로 분석
4. **상위 1%만 GA4를 활용**하여 데이터 기반 운영

### 3-5. 애드센스 자동광고 OFF 권장
- 수동 배치가 수익 효율 높음
- 자동광고는 레이아웃을 해침

## PART 4. 워드프레스 세팅 A to Z

### 4-1. 호스팅 선택
| 호스팅 | 특징 | 가격 |
|--------|------|------|
| **Cloudways** | 관리형, 편리 | 월 $11~14 |
| **Vultr** | VPS, 가성비, 10GB 저장, 500GB 트래픽 | 월 $2.5~ |
| **카페24** | 국내, PHP 7.4 한정 | 월 500~5,500원 |

- **추천**: Vultr 또는 Cloudways (속도, 안정성, PHP 8.2 지원)
- 서버 스펙: **2GB RAM** 최소, **4GB** 권장
- 호스팅 3가지 요소: 서버(웹호스팅), 도메인, SSL인증서
- 참고: Vultr, Linode 추천 / DigitalOcean도 대안

### 4-2. 도메인 설정
- 가비아(gabia.com)에서 도메인 구매
- .com, .co.kr: 연 1~2만원 / .xyz, .info: 연 500~5,000원
- DNS 설정: A레코드 → 서버 Public IP 연결
- CNAME 설정: 서브도메인 연결
- SSL 인증서: **필수** (https → SEO 가점)
- Cloudways: DOMAIN MANAGEMENT → ADD DOMAIN → SSL Certificate 설치

### 4-3. 테마 설정 - GeneratePress
- **GeneratePress + GP Premium** 조합 추천
- 경량, 빠른 속도, SEO 최적화, 커스터마이징 용이
- 세팅 순서:
  1. GeneratePress 설치 → GP Premium 활성화
  2. Site Library에서 **2컬럼 레이아웃** 선택 (PC 2컬럼, 모바일 1컬럼)
  3. **Container Width**: 790~800px
  4. **Header**: 간결하게
  5. **Sidebars**: 2컬럼이면 사이드바 1개
  6. **Blog Content**: Featured images 설정, Archives 정리
  7. Domain Privacy 설정으로 개인정보 보호

### 4-4. 필수 플러그인 8종

| 플러그인 | 용도 |
|----------|------|
| **Rank Math (Pro)** | SEO 최적화 (메타태그, XML 사이트맵, 스키마) |

### 4-5. 이미지 최적화 (페이지 속도)
- **Imagify**: 이미지 압축
- **Converter for Media**: WebP 자동 변환
  - Conversion strategy: Optimal
  - Supported output formats: WebP
  - Bulk Optimization으로 기존 이미지 일괄 변환
  - Cloudways: APPLICATION SETTINGS > WEBP REDIRECTION 활성화
- 목표: PageSpeed Insights (pagespeed.web.dev) **90점 이상** (98~100점 달성 사례)

### 4-6. Ad Inserter 광고 배치
- Block 1~16까지 광고 코드 등록
- 위치 설정 옵션:
  - **Before Content**: 본문 상단
  - **After Content**: 본문 하단
  - **Before/After Paragraph**: 특정 단락 전후
  - **Before/After Image**: 이미지 전후
  - **Between Posts**: 글 목록 사이
- 2컬럼 1사이드바, RPM 기준으로 상단 광고 집중

### 4-7. Rank Math SEO 설정
- 기본 모듈 ON: 404 모니터, 리다이렉션, 스키마, 사이트맵, SEO 분석
- 글 작성 시 Focus Keyword → SEO 점수 50점 이상 목표
- 글 작성 체크리스트:
  1. 제목에 Focus Keyword 포함 (2023년 8월 이후 기준)
  2. H2 태그에 키워드 포함
  3. 메타 디스크립션 직접 작성
  4. 본문 2000자 이상
  5. 이미지 Alt 태그에 키워드 포함
  6. 내부링크(관련글) + 외부링크(출처) 포함

### 4-8. WP Rocket 설정
- CACHE: 페이지 캐싱 활성화
- File Optimization: CSS/JS 최적화
- Media: 지연 로딩
- Preload Cache: 사전 캐시
- Database: DB 정리
- Heartbeat: 불필요한 요청 제한

### 4-9. ads.txt 설정
- File Manager 또는 FTP로 루트에 ads.txt 파일 생성
- 구글 애드센스 정책 필수 준수

### 5-2. Google Search Console 등록 절차
1. Search Console 접속 → 속성 추가
2. URL 접두어 방식으로 사이트 URL 입력 (http/https 구분 주의, SSL 적용 시 https)
3. HTML 태그 방식으로 소유권 확인
4. **사이트맵 제출**: `/sitemap.xml` 과 `/rss` 모두 등록
5. URL 검사 도구로 색인 요청
6. 개별 URL도 수동으로 색인 요청 가능

### 5-3. Daum/다음 등록 (티스토리용)
- 블로그 등록: URL, 사이트 이름, 설명, 카테고리 입력
- 핀(PIN) 인증: +8~12자리 → 1~수일 소요
- 개별 글 URL도 수동 등록 가능
- 다음 웹마스터 도구: webmaster.daum.net

### 5-4. 네이버 Search Advisor
- 사이트 소유 확인: HTML 파일 업로드 or HTML 태그
- 사이트맵, RSS 제출
- robots.txt 확인

### 5-5. Bing Webmaster Tools
- Google Search Console과 연동 가능 (자동 import)
- 사이트맵 제출: `domain/sitemap.xml` 또는 `domain/sitemap_index.xml`

  - WPCode → Header에 Clarity 코드 삽입
- Bing 등록 시 **Yahoo 검색도 자동 적용**

## PART 6. 글쓰기 전략

### 6-1. 글의 4가지 핵심 요소
1. **제목**: 키워드 포함, 호기심 유발, 40~60자
2. **부제목 (H태그)**: H2, H3, H4 계층 구조
3. **본문**: 1,500자 이상 권장, 가독성 중심
4. **이미지/동영상**: 최소 3~4개, 15~20줄마다 삽입

### 6-2. H태그 구조 (SEO 핵심)
```
H1: 글 제목 (자동 생성, 1개만)
  H2: 대주제 1
    H3: 소주제 1-1
    H3: 소주제 1-2
  H2: 대주제 2
    H3: 소주제 2-1
      H4: 세부 내용
```
- 각 H태그에 키워드/연관 키워드 포함
- 1번 대주제, 2번 소주제, 3번 세부내용 구조 유지

### 6-3. 본문 작성 규칙
- **분량**: 1,500자~3,000자 (SEO 최적)
- **가독성**: 15~20줄마다 이미지/소제목 삽입, 3줄 이상 연속 텍스트 지양
- **링크**:
  - 내부링크(자기 글): Ctrl+K로 삽입, 앵커 텍스트에 키워드
  - 외부링크(권위 있는 사이트): `target="_blank"` (새 탭 열기)
- **키워드 밀도**: 자연스럽게 3~5회 포함
- **첫 문단**: 키워드를 반드시 포함
- **이미지**: Alt 태그에 키워드, 500KB 이하 압축 (iLoveIMG 활용)

### 6-4. 제목 작성 전략
- **30자~50자** 추천 (검색 결과 노출 최적)
- 호기심 유발 요소 포함
- "~방법", "~총정리", "~비교", "~추천" 등 검색 의도 반영
- 숫자 포함 시 CTR 상승 (예: "2024년 TOP 5", "3가지 방법")

### 6-5. 썸네일 전략
- 크기: **500x500** 또는 **1200x628**
- 제작 도구: **미리캔버스** (무료), PPT도 가능
- 핵심: **텍스트 포함**, 호기심 유발, 대비 색상, 큰 글자

### 6-6. 30일 글쓰기 플랜
- **1일 1포스팅 원칙** (최소 30일)
- 주제 & 카테고리 1개 = 30일분 키워드 미리 선정
- 카테고리 1개 = 최소 20편
- 글감 1 = 30일분 키워드 리스트 → 글감 1 = 카테고리 (최소 3개 카테고리)
- 날짜별 예약 발행 활용

### 6-7. ChatGPT 활용 글쓰기
1. 주제 관련 키워드 30개 추출 요청
2. 각 키워드별 3000자 SEO 최적화 글 요청
3. **주의**: AI 글 100% 복붙 금지, 반드시 20~30% 이상 수정/재작성
4. 1일 15~20편 발행 가능 (AI + 수정 조합)
5. GPT-4 또는 GPT-3.5 활용, Plugins 기능 활용

### 6-8. 7대 카테고리 추천
1. **건강**: 질병/증상/영양제/다이어트
2. **금융**: 대출/보험/투자/세금
3. **생활**: 생활꿀팁/정리수납/가전리뷰
4. **IT/테크**: 스마트폰/앱/프로그램
5. **여행**: 국내/해외/맛집/숙소
6. **교육**: 자격증/입시/학습법

## PART 7. 키워드 리서치 (수익의 핵심)

### 7-1. 키워드 선정 원칙
- **CPC가 높은 키워드** = 수익이 높은 키워드
- 검색량과 경쟁도의 균형 찾기
- 키워드 3가지 유형:
  1. **정보성 키워드**: "~방법", "~하는 법"
  2. **상업성 키워드**: "~추천", "~비교", "~가격"
  3. **거래성 키워드**: "~구매", "~신청"
- VIEW, 블로그, 카페, 지식인 탭에서 경쟁도 확인

### 7-2. 1000만원 키워드 발굴 공식
1. **대주제 선정**: 건강, 금융, 부동산, 보험, IT
2. **40~60자 제목** 작성 (검색 노출 최적화)
3. **복합 키워드** 활용: "~방법", "~후기", "~비교", "~가격"

**예시 - 부동산 주제:**
- 대분류: 부동산 → LH 임대, 청약, 세금, 대출
- 세분화: "LH 임대 >> 신혼부부, 조건 자격" / "청약 >> 일정, 자격조건"
- 제목: "2024년 LH 신혼부부 임대아파트 조건 자격 총정리"

### 7-3. 키워드 도구 6종

| 도구 | 용도 | URL |
|------|------|-----|
| **Google Trends** | 트렌드 분석 | trends.google.co.kr |

  - 예: tistory.com 검색 시 2900만개 키워드, 1200만개 유기적 트래픽
- **백링크 분석**: 경쟁 사이트의 백링크 구조 파악
- CSV 내보내기로 키워드 리스트 관리
- Guru 요금제 추천 (7일 무료 후 해지 가능)

### 7-5. 정부 사이트 활용 키워드 발굴
- **정책브리핑** (korea.kr): TOP 50 정책 키워드 → 블로그 소재
  - "20OO년 ~정책", "ISA", "청년도약계좌" 등
- **24시 정부민원** (gov.kr): 복지/지원금 키워드
- **복지로** (bokjiro.go.kr): 복지 혜택 키워드
- **서울미디어허브** (mediahub.seoul.go.kr): 서울시 정책
- **지자체 블로그**: 서울시, 경기도 등 공식 블로그에서 소재 발굴
- 24시, 복지로, 지자체 블로그에서 키워드 365일 수급 가능

### 7-6. 상위 0.1% 키워드 전략

- 키워드 카테고리별 분류: OOO 키워드(건강, IT, 금융 등) + 접미사(방법, 추천, 비교 등)

## PART 8. 광고 배치 최적화

### 8-1. 광고 배치 기본 원칙
- 모바일 트래픽이 **58% 이상** → 모바일 최적화 필수
- PC와 모바일 동시 테스트

### 8-2. 최적 광고 배치 위치
| 위치 | 설명 | 중요도 |
|------|------|--------|
| **글 상단 (본문 시작 전)** | 가장 높은 CTR | ★★★★★ |
| **본문 중간** | 2번째로 높은 수익 | ★★★★ |
| **글 하단 (본문 종료 후)** | 보조 수익 | ★★★ |
| **사이드바** | PC 전용, 보조 | ★★ |

### 8-3. 배치 조합별 전략
1. **기본형**: 상단 1개 + 하단 1개
2. **수익 극대형**: 상단 2개(PC) / 1개(모바일) + 중간 1개 + 하단 1개
3. **안전형**: 상단 1개 + 중간 After Paragraph(2번째 단락 후) + 하단 1개
4. **사이드바 포함**: 위 조합 + 사이드바 고정 광고 (PC만)

### 8-4. 광고 실험/최적화
- 애드센스 > 실험 기능: 1일, 2일, 5일, 10일, 30일, 1년 단위 테스트
- RPM 기준으로 광고 위치/크기/형식 비교
- RPM이 10 이하면 배치 변경, 10~20이면 양호, 20 이상이면 우수

## PART 10. SEO 심화 전략

### 10-1. SEO의 4대 축
1. **기술적 SEO**: 사이트 속도, 모바일 최적화, 크롤링 허용, SSL
2. **메타태그 SEO**: title, description 최적화
3. **콘텐츠 SEO**: 양질의 콘텐츠, 키워드 밀도, H태그 구조
4. **링크 SEO**: 내부링크, 외부링크, 백링크

### 10-2. 구글 E-E-A-T 기준
- **Experience (경험)**: 실제 경험 기반 콘텐츠
- **Expertise (전문성)**: 해당 분야 전문 지식
- **Authoritativeness (권위)**: 사이트/저자의 신뢰도
- **Trust (신뢰)**: 가장 중요한 핵심 요소
- **실천법**: About Us 페이지, 저자 프로필, LinkedIn 연동

### 10-3. 내부링크 전략
- 관련 글끼리 상호 링크 연결
- **앵커 텍스트**에 키워드 포함
- 1글당 최소 2~3개 내부링크 권장
- 카테고리 내 글끼리 연결하면 크롤링 효율 상승

### 10-4. 메타 디스크립션 최적화
- 100~120자 권장
- 핵심 키워드 포함
- 행동 유도 문구 포함 ("~알아보세요", "~확인하세요")
- 각 글마다 고유한 디스크립션 작성

### 10-5. 색인 관리
- URL 검사 도구: 글 발행 후 즉시 색인 요청
- `site:도메인` 검색으로 색인 상태 확인
- Google Index API 활용 (워드프레스)
- 30일 이내 발행 글은 URL 수동 제출 권장

### 10-6. 구글 SEO 히스토리 핵심
- 구글의 핵심: **페이지랭크** (백링크 기반 신뢰도 평가)
- 검색엔진은 "좋은 콘텐츠"를 판별하기 위해 **링크의 질과 양** 분석
- SEO 본질: "검색하는 사람에게 가장 유용한 정보를 제공"
- Black Hat SEO(스팸 링크, 키워드 스터핑) → 패널티
- White Hat SEO(양질 콘텐츠, 자연스러운 링크) → 장기적 성장

## PART 11. 백링크 & 외부유입

### 11-1. 백링크란?
- 다른 사이트에서 내 사이트로 연결되는 링크
- **구글 SEO 순위 결정의 핵심 요소**
- 백링크의 질 > 양 (고품질 사이트에서의 1개 > 저품질 100개)
- DA(Domain Authority) 높은 사이트에서의 백링크가 가치 있음

### 11-2. 백링크 품질 확인 도구
- **Ahrefs** (ahrefs.com): DR, UR 지표 확인, Backlink Checker (1일 99회)
- **SimilarWeb**: 트래픽 소스, 참조 도메인 확인
- **Semrush**: 백링크 Analytics, Overview, Anchors 분석 (1일 99회)

### 11-3. 백링크 구축 전략
1. **커뮤니티 활용**: 보배드림, FM코리아, SLR, 82Cook, MLB파크 등 고DA 사이트
   - 글 작성 시 자연스럽게 링크 삽입
   - `nofollow` 링크도 트래픽 유입에는 효과 있음
2. **위키백과**: 위키 문법으로 링크 삽입 (높은 DA)
   - `[URL 텍스트]` 형식
3. **네이버 블로그/카페**: m.blog.naver.com 활용
   - SEO 효과는 30일~1개월 후 나타남
4. **게스트 포스팅**: 관련 블로그에 기고
5. **SNS 공유**: 페이스북, 트위터 등에 URL 공유
6. **커뮤니티 사이트 리스트**: 네이버카페, SLR클럽, 뽐뿌, 클리앙, 디시인사이드, FM코리아, 일베, 루리웹 등

### 11-4. 네이버 블로그 외부유입 전략
- 네이버 블로그에서 구글 SEO 사이트로 유입 유도
- 블로그 글에 자연스럽게 워드프레스/티스토리 링크 삽입

### 11-5. 경쟁사 분석 기법

## PART 12. 다중 사이트 전략 (스케일업)

### 12-1. 2차 도메인(서브도메인) 활용
- 메인 도메인 ABC.com에서 서브도메인 생성
  - notice.ABC.com / marketing.ABC.com
- 각 서브도메인 = 독립 블로그 운영 가능
- 메인 도메인의 **DA**가 서브도메인에 전이

### 12-2. 도메인 구매 전략
- 가비아에서 구매
- **.com / .co.kr**: 신뢰도 높음 (연 1~2만원)
- **.xyz / .info**: 저렴하지만 신뢰도 낮음 (.xyz보다 .com 추천)
- **핵심**: 도메인 이름에 키워드 포함 (영문)
- 서브도메인: 3~5개 운영으로 **수익 10배** 가능

### 12-3. 다중 사이트 운영법
- 1차: 메인 도메인에 20편 발행 → 색인 안정화
- 2차: 서브도메인 개설 → 각 10~15편 발행
- **주의**: 같은 서버에서 운영 시 IP 동일 → WHOIS 정보 확인
- 3가지 연결 방법: 내부링크, 리다이렉트, 독립 운영

### 12-4. 100편 빠르게 채우는 법
- 1일 1편 x 100일 → 너무 느림
- **전략**: 1일 3~6편 (AI 활용) → 1~2개월 내 100편 달성
- 핵심: 30편 이상 발행 후 검색엔진 URL 일괄 제출
- 실전 사례: 150편 → 7개월 40만 조회, 259만원 수익

### 12-5. 서비스 사이트 운영
- 수익 사이트(애드센스) + 서비스 사이트 병행 구조
- 서비스 사이트 예시: biz.postincome.co.kr, mytravelnotes.co.kr
- 서비스 사이트: 100% 자체 콘텐츠 + 80% 수익형 혼합
- DA 높은 메인에서 서비스 사이트로 링크 주입

- 비교형: "A vs B"
- 가이드형: "완전 가이드", "A to Z"
- 리뷰형: "솔직 후기", "장단점 비교"
- PPT/인포그래픽 활용

### 14-3. 수익 극대화 공식
- 글 하나에 여러 내부링크 → 페이지뷰 극대화
- 버튼형 CTA (HTML/CSS) → 제휴 링크 클릭율 UP
- 시리즈물 구성: 1편 → 2편 → 3편 자연스러운 이동

## PART 15. AI 쇼츠 (유튜브 추가 수익)

### 15-1. 유튜브 쇼츠 수익 구조
- **YPP (파트너 프로그램)**: 구독자 1,000명 + 쇼츠 1,000만 조회
  - 쇼츠 1만 조회당 200~230원 (2024년 기준 150~180원)
- **제휴 마케팅**: 10만 뷰당 150만원 가능
- **브랜드 협찬**: 채널 성장 후 직접 광고 계약

### 15-2. AI 쇼츠 제작 시스템
- **AI 도구 활용**: ChatGPT(대본), TTS(음성), AI 영상 생성
- **후킹 기법** (첫 3초):
  - 호기심: "이거 몰랐어요?"
  - 감정: "3초 후 반전이..."
  - 대립: "A vs B, 어떤 게 맞을까?"
- **최적 길이**: 15~45초, 3분 미만이 80% 완시율
- **인기 장르 TOP 5**: 정보/리뷰, 감성, 밈, 실험, 브이로그

### 15-3. 쇼츠 대본 작성 구조
1. 첫 3초: 후킹 (호기심/감정/대립)
2. 본문: 핵심 정보 전달
3. 마무리: CTA (좋아요/구독/댓글)
- TTS 속도: 1.2배속 추천 (1.4배속은 빠름)
- BGM: 분위기에 맞는 배경음악 필수

### 15-4. 수익화 타임라인
- 1~3개월: 매일 1~2개 업로드, 100개 콘텐츠 축적
- 3~6개월: YPP 달성, 광고 수익 시작
- 6개월+: 제휴마케팅, 브랜드 협찬 추가
