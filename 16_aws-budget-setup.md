# 16. AWS 예산 알림 설정

## 개요
AWS 비용이 임계값 초과 시 이메일 알림을 받아 예상치 못한 과금을 방지

---

## Lightsail $20 플랜 기준 비용 구조

| 항목 | 비용 |
|---|---|
| 인스턴스 | $20/월 |
| 데이터 전송 포함량 | 4TB/월 |
| 도메인 | ~$1/월 |
| **예상 고정 비용** | **~$21/월** |

> 데이터 전송 4TB는 일반적인 자동화 규모에서 초과하기 어려움

---

## 설정 절차

### 1. AWS Cost Management 접속

**AWS 콘솔 → 상단 검색 → "Budgets" 검색 → 예산 생성**

### 2. 예산 유형 선택

- 템플릿 사용(신속) 선택
- **월별 비용 예산** 선택

### 3. 예산 설정

| 항목 | 값 |
|---|---|
| 예산 이름 | `monthly-cost-alert` |
| 예산 금액 | `$30` |
| 이메일 수신자 | 관리자 이메일 (최대 10개) |

### 4. 알림 조건 (자동 설정됨)

| 조건 | 기준 금액 |
|---|---|
| 실제 지출 85% 도달 | $25.5 |
| 실제 지출 100% 도달 | $30 |

### 5. 생성 완료 확인

예산 개요 화면에서 상태 **"정상"** 확인

---

## 주의사항

- 예산 알림은 알림만 발송, 자동 차단 없음
- $30 초과 알림 수신 시 AWS 콘솔에서 비용 항목 직접 확인 필요
- Lightsail → 네트워킹 탭에서 데이터 전송 사용량 모니터링 가능
